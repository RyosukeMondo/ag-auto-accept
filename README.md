# Ag-Accept

A lightweight tool to automatically accept antigravity windows.

## Installation

You can install this tool directly from GitHub using `pipx`. This ensures it runs in an isolated environment while being globally accessible.

```bash
pipx install git+https://github.com/RyosukeMondo/ag-accept.git --force
```

## Usage

Once installed, you can run the tool using the command:

```bash
ag-accept
```

## Features

- lightweight GUI
- Dark/Light mode support
- Configurable monitoring interval
- Dual Modes:
    - **IDE Mode**: For regular Antigravity IDE usage (sends Alt+Enter).
    - **AgentManager Mode**: For watching AgentManager windows (clicks "Accept" button).
- Auto-accept functionality for "Antigravity" windows
